	\subsection{安装必要依赖}
	首先执行更新并安装基础包：
	
	\begin{lstlisting}[language=bash]
		sudo apt update && sudo apt upgrade -y
		sudo apt install build-essential git -y
	\end{lstlisting}
	
	\subsection{修改配置文件}
	编辑 \texttt{/etc/example.conf} 文件，确保以下参数已更改：
	
	\begin{lstlisting}
		# Example Configuration
		SERVER_PORT=8080
		ENABLE_LOG=true
	\end{lstlisting}
